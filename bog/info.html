About

Mission Statement

We are hoping to provide the aggressive inline community with consistent and understandable information regarding aggressive inline rollerblading.

There is lots of material on the internet but it is all over the place and we're hoping to tie it together in one source.

We are aiming to educate new, and existing, rollerbladers and expose this sports complexity.

Aggressive inline deserves to be widely recognized and brought back to it's glory days so spread the word and get people hooked!

If we manage to inspire at least one person out there then this project is a success!

Through the progression of this project and our personal development we are hoping to tailor the material for anyone who is willing to teach others about Aggressive Inline. If you can use the material to teach other people, please do so. We would greatly appreciate if you could reference our website and our project if you use our material (technically and legally you must due to the CC BY-NC-SA License).

Versioning

Versioning of the website is provided because the creator of the scripts is a coding geek.

It is also there for your reference to check if the website has been updated since the last time you checked (so remember the number if this is something you’re interested in).

The format of versioning is "a.b.c D" where:

a = major version release (i.e. huge overhaul or something very new or different)

b = minor version release (i.e. some template changes, bug fixes, script related stuff)

c = minor build version (i.e. new grinds, spelling fixes, small visual or contextual changes – this number will increase most rapidly)

D = version description (i.e. BETA, or Final release, Special release, or whatever the creator comes up with)

We are not keeping a log, or version history.

Frequently Asked Questions

Why skateyeg.com?

The project, initially, was to become a website for general skatepark information for the Edmonton Area with inclusion of sections for all sorts of action sports. It quickly became evident that this project was way too large for one person to accomplish and it sat dormant for a while, with the domain registered already and a couple initial versions of websites built. Edmonton is identified by the YEG airport code which is commonly used on social media as YEG... something... to tag things that are happening in the city. Skate, well, because of skateparks. It nicely translated to the rollerblading project we are working on so the domain remained as is (we do own http://bookofgrinds.com as well but are staying with the original domain as the primary).

Why is this content free?

The Director of this project believes in free education, and that information should be available to public at no cost for growth and learning - no matter if you have the money to consume it, or not. We also want to promote rollerblading and give people material they can use to explore and learn the sport from a consistent source. Any income from donations or other support that we generate from this project is directly invested to cover online maintenance fees and support our Junior Bladers Skate Team Program in Edmonton, AB - Canada.

Will there be a video game?

Wouldn't that be something?! Video games are a huge undertaking. We would love to have a video game of some sort out, one day. But can't really say for sure if it will ever happen.

Will there be different language options?

Eventually, yes. This will require proper translations of each page and it's content. In meantime you may try using Google Translate (a free translator) to look up the content - it might not be perfect but it's better than nothing.

Will there be right leg dominant grind previews?

It is very time consuming to create the grinds "the other way" so for consistency we are using left leg dominant previews. There is no immediate plan to create previews for right leg dominant grinds however you may find one or two that are shown with the right leg as the primary leg.
